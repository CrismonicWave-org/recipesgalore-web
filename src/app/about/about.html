About RecipesGalore

An open source project dedicated to making recipes accessible, shareable, and easy to use for everyone.

Our Vision

RecipesGalore is built on a unique architecture that leverages the power of open source and modern web technologies:

📚 GitHub as Database

Recipe content lives in GitHub repositories as Markdown files with YAML front matter. This provides version control, transparent governance through pull requests, and an auditable history of every recipe change.

🚀 Firebase Hosting & Services

The Angular web app is deployed to Firebase, providing fast global CDN delivery, authentication services, and scalable infrastructure without the complexity of managing servers.

📱 Cross-Platform Architecture

Built with Angular for web and planned IONIC/Capacitor for iOS and Android. One codebase, multiple platforms, with offline-first capabilities powered by versioned release bundles.

🔒 Security First

GitHub Advanced Security (GHAS) ensures code quality and security. Protected main branches, required reviews, and automated security scanning keep the project safe and reliable.

We believe recipes should be free, community-driven, and accessible to everyone. By combining GitHub's collaboration tools with modern web frameworks, we're creating a recipe platform that's transparent, maintainable, and built for the long haul.
